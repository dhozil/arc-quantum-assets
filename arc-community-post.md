# Quantum Asset Hub - RWA Tokenization Demo on Arc Network

## Project Overview

Quantum Asset Hub is a demo application built for the Arc Network testnet program. It demonstrates the concept of Real World Asset (RWA) tokenization using blockchain technology. The platform showcases how traditional assets like real estate, energy projects, commodities, and treasury instruments can be tokenized and traded on-chain.

## Purpose & Utility

**What is RWA Tokenization?**
Real World Asset tokenization is the process of representing physical or traditional financial assets as digital tokens on a blockchain. This technology enables:
- Fractional ownership of high-value assets
- 24/7 trading without intermediaries
- Global accessibility to investment opportunities
- Transparent and immutable transaction records
- Lower barriers to entry for investors

**Why Arc Network?**
Quantum Asset Hub is built on Arc Network because of its:
- Fast transaction confirmations
- Low gas fees making micro-investments viable
- Energy-efficient consensus mechanism
- Developer-friendly ecosystem
- Growing community and dApp ecosystem

## Features Implemented

### 1. Unique UI/UX Design
- **Hexagonal Buttons**: Custom hexagonal shape for all interactive elements
- **Particle Background**: Animated particle system with dynamic connections
- **Neon Glow Effects**: Cyberpunk-inspired neon lighting effects throughout
- **3D Card Tilt**: Asset cards respond to mouse movement with 3D perspective
- **Glass Morphism**: Frosted glass effect for modern aesthetic

### 2. Asset Dashboard
- **Category Filtering**: Filter assets by Real Estate, Energy, Commodity, Treasury
- **Real-time Statistics**: Track balance and total assets owned
- **Asset Cards**: Display price, APY, value, and 24h change for each asset
- **Interactive Charts**: Price history visualization for each asset

### 3. Trading Interface
- **Full Trading Page**: Complete buy/sell order management
- **Order Book**: View and manage market and limit orders
- **Price Charts**: Historical price data with visual charts
- **Asset Details**: Complete information about each tokenized asset

### 4. Smart Contract Integration
The platform integrates with three core smart contracts:
- **QuantumRWA**: Asset tokenization and management
- **QuantumFactory**: Deploy new RWA tokens
- **QuantumMarketplace**: Decentralized trading and order matching

### 5. Wallet Integration
- **MetaMask Support**: Seamless wallet connection
- **Auto Network Switching**: One-click switch to Arc Network
- **Balance Display**: Real-time USDC balance tracking
- **Transaction Notifications**: Toast notifications for all actions

### 6. Admin Panel
- **Owner Functions**: Mint tokens to specific addresses
- **Deploy New Assets**: Create new RWA tokens through factory contract
- **Asset Management**: Configure asset parameters (APY, total value, etc.)

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Blockchain**: ethers.js v6 for contract interaction
- **Styling**: Tailwind CSS with custom CSS effects
- **Animations**: Framer Motion for smooth transitions
- **Network**: Arc Network testnet (Chain ID: 316450)

## How to Use

1. Connect your MetaMask wallet
2. Switch to Arc Network (automatic prompt)
3. Browse assets by category
4. Click "Trade" to access trading interface
5. Click asset card for mint/redeem operations
6. Use Admin Panel (owner only) for deployment and minting
